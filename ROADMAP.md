# Scope PM — Feature Roadmap

## High-Impact Features

### 1. Cross-Interview Synthesis
- [ ] Cluster themes/pain points across all interviews in a project
- [ ] Rank by frequency ("7 of 10 users mentioned X")
- [ ] Surface consensus vs. outlier insights
- [ ] Synthesis dashboard view per project
- [ ] AI-generated summary of top patterns

### 2. Prioritization Engine
- [ ] Impact vs. effort matrix view
- [ ] Auto-suggest priority using interview frequency + competitor gaps
- [ ] Drag-and-drop override for manual ranking
- [ ] Scoring model (customizable weights)

### 3. Spec Export / Handoff
- [ ] Export specs as Markdown
- [ ] Export specs as PDF
- [ ] Linear integration
- [ ] Jira integration
- [ ] Notion integration

### 4. Team Collaboration
- [ ] Multi-user projects (invite by email)
- [ ] Roles: viewer, editor, admin
- [ ] Comments on specs and interviews
- [ ] @mentions and activity feed per project
- [ ] Shared project dashboard

### 5. Interview Recording + Transcription
- [ ] Upload audio/video files
- [ ] Auto-transcription (Whisper / Deepgram)
- [ ] Paste Zoom/Google Meet recording link
- [ ] Timestamped transcript with playback

### 6. Feedback Loop / Outcome Tracking
- [ ] Mark specs as shipped
- [ ] Link shipped specs back to source interviews
- [ ] Track whether pain points were resolved
- [ ] Outcome metrics dashboard

### 7. Customer Persona Builder
- [ ] Auto-generate personas from interview data
- [ ] Role, goals, frustrations, key quotes
- [ ] Editable persona cards
- [ ] Share personas across projects

---

## Table Stakes (Must-Haves)

### 8. Search & Filtering
- [ ] Global search across interviews, specs, competitors
- [ ] Filter by status, priority, date range
- [ ] Search within interview transcripts

### 9. Bulk Interview Upload
- [ ] Upload multiple transcripts at once
- [ ] Batch AI analysis with progress tracking

### 10. Spec Versioning / History
- [ ] Track edits over time
- [ ] Diff view between versions
- [ ] Restore previous versions

### 11. Dashboard Charts
- [ ] Interviews over time
- [ ] Spec status breakdown (pie/bar chart)
- [ ] Analysis completion trends

### 12. Email Notifications
- [ ] Alert when AI analysis completes
- [ ] Notify on spec status changes
- [ ] Weekly digest of project activity

### 13. Data Export
- [ ] CSV export of interviews, specs, competitors
- [ ] PDF export for reporting
- [ ] Bulk export per project
